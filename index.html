Diferencias entre Git y Github.

Git es un sistema de control de versiones que permite a varios desarrolladores trabajar en un mismo proyecto de manera simultánea. Es una herramienta gratuita y de código abierto. Git es útil para: Seguir el historial de cambios en un proyecto Revertir el código a una versión anterior Inspeccionar los cambios de archivos Trabajar en equipo de manera simultánea Administrar la separación entre los cambios de diferentes desarrolladores
GitHub es una plataforma en la nube que permite a los desarrolladores almacenar, compartir y trabajar en código. Se basa en el sistema de control de versiones Git. Qué se puede hacer en GitHub. Almacenar y administrar código Seguir y administrar cambios en el código Trabajar en proyectos colaborativos Descargar programas o aplicaciones Aportar mejoras a programas o aplicaciones Discutir cuestiones en foros temáticos Explorar repositorios, temas, código, usuarios y organizaciones
